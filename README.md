# 🔍 Azure AI Search: Knowledge Mining & Inteligência Cognitiva by NETZ

## 🚀 Resumo do Projeto
Este repositório documenta a implementação de uma solução de **Mineração de Conhecimento** e análise de dados não estruturados utilizando a plataforma Azure. O foco foi transformar arquivos brutos (PDFs e Imagens) em um índice inteligente pesquisável, aplicando técnicas de OCR e Processamento de Linguagem Natural (NLP).

## 🛠️ Tecnologias Utilizadas
- **Azure Blob Storage:** Armazenamento dos documentos brutos.
- **Azure AI Services:** Enriquecimento cognitivo (OCR e Extração de Frases-chave).
- **Azure AI Search:** Motor de busca semântica e indexação.

## 📈 Insights Estratégicos (Visão NETZ)
1. **Poder do OCR:** A capacidade de extrair texto de imagens (como visto em `/images/resultado_mineracao_netz.png`) é um diferencial para digitalizar infraestruturas legadas.
2. **Busca Semântica:** A implementação do classificador semântico permite que o conhecimento técnico seja encontrado pela intenção, e não apenas por palavras-chave exatas.
3. **Escalabilidade:** Esse pipeline permite que a NETZ mantenha uma base de conhecimento automatizada, reduzindo o tempo de resposta técnica.
